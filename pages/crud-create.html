Create a New TenantLens Record

This page explains how renters would submit a new report, review, or lease-risk signal.

How Create Would Work

Users fill a structured form with property details, issue category, incident date, and supporting notes. Optional evidence links can be added for validation.

  • Input fields: building name, address, landlord/management, category, severity
  • Validation: required fields, duplicate-check against existing reports
  • Submission: stores a new report and triggers moderation review
