# Tenant Intelligence Platform (Static Frontend)

This project is a static frontend prototype for the Tenant Intelligence Platform.  
It demonstrates the look and feel of the product vision using HTML, CSS, and minimal JavaScript.

## Product Direction

The platform is tenant-first and AI-powered. Its purpose is to reduce rental fraud, hidden fees, and poor lease decisions by giving renters actionable intelligence before they sign.

## Homepage Coverage

Current homepage includes:
- Futuristic peach-themed UI with subtle animations
- Priority search experience as the main interaction
- Project purpose and core value messaging
- Key MVP feature cards
- Recent reviews section (dummy data)
- Tenant testimonials section (dummy data)

## AI Additions Included

The homepage and product documentation now include these new AI modules:

1. **Natural Language Search**  
   Users can search in plain language (for example: budget + risk + maintenance quality in one query).

2. **Renter Fit Assistant**  
   Generates personalized property recommendations using renter preferences and constraints.

3. **Negotiation Copilot**  
   Suggests negotiation points and follow-up questions from lease risks and listing signals.

4. **Deposit Recovery Assistant**  
   Helps renters prepare dispute workflows with evidence checklists and draft response language.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (minimal behavior only)
